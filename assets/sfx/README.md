## Format

- **Format**: MP3 or WAV
- **Duration**: Short (<1s), Medium (1-2s), Long (2-4s)
- **Quality**: Any quality works, 128kbps MP3 recommended
- **Volume**: Normalize to consistent levels

## Quick Setup

For testing, you can use system sounds or generate simple beeps:

```bash
# macOS - use system sounds as placeholders
cp /System/Library/Sounds/Ping.aiff cat_short.mp3
cp /System/Library/Sounds/Purr.aiff cat_medium.mp3
```

Or download from Pixabay and rename to match the pattern above.
